LIS（最长上升子序列问题）

LIS是DP问题中的一个重要模型，我们先来看一下原始问题：

原始问题

问题描述

给定一个长度为 N 的数列，求数值严格单调递增的子序列的长度最长是多少。

子序列 是由数组派生而来的序列，删除（或不删除）数组中的元素而不改变其余元素的顺序。例如，[3,6,2,7] 是数组 [0,3,1,6,2,2,7] 的子序列，而 [0,1,2,7]是原数组的最长上升子序列，长度为4。

DP思路

状态表示

我们用 f[i] 表示以 nums[i] 结尾的上升子序列的最大长度 ，显然 max(f[1 ~ n]) 即为所求（数组下标从1开始，长度为n）

状态计算

对于 f[i] 代表的这个序列，我们以该序列倒数第二个元素分类：倒数第二个元素分别为num[i-1]...num[1]或只有一个元素，因此得到以下情况：

  • 倒数第二个元素是num[i-1] （要求num[i] > num[i-1]）：f[i] = f[i-1] + 1

  • ......

  • 倒数第二个元素是num[1] （要求num[i] > num[1]）：f[i] = f[1] + 1

  • 只有一个元素：f[i] = 1

故得到状态转移方程：f[i] = max(1,f[j] + 1)（j∈(1..i−1)且nums[i] > nums[j]）

时间复杂度

O(n2)

示例代码

#include<iostream>
#include<cstring>
#include<algorithm>

using namespace std;

const int N = 1010;
int nums[N],f[N];
int n,res;

int main()
{
    cin >> n;
    for(int i = 1;i <= n;i++) cin >> nums[i];
    for(int i = 1;i <= n;i++)
    {
        f[i] = 1;
        for(int j = 1;j < i;j++)
            if(nums[i] > nums[j]) f[i] = max(f[i],f[j] + 1);
        res = max(res,f[i]);
    }
    cout << res;
    return 0;
}

扩展问题

1.正反LIS

  • 登山

题目分析

这道题的题意大致如下：

我们可以任意选择起点 a --（上山）--最高点 b --（下山）--终点 c ，可以跳着选择浏览景点，以期望达到浏览景点数最大，如图。

也就是我们需要任意选择最高点 b ，使得以 b 为终点的上升子序列长度 + 以 b 为起点的下降子序列长度最大，而下降子序列就是反向的上升子序列。

而这二者互不影响（ b 左侧选择的景点不影响 b 右侧选择的景点），因此为求整体最大，只需要求二者各自的最大值即可

示例代码

#include<iostream>
#include<cstring>
#include<algorithm>

using namespace std;

const int N = 1010;
int f[N],g[N],nums[N];
int n,res;

int main()
{
    cin >> n;
    for(int i = 1;i <= n;i++) cin >> nums[i];
    for(int i = 1;i <= n;i++)
    {
        f[i] = 1;
        for(int j = 1;j < i;j++)
            if(nums[i] > nums[j]) f[i] = max(f[i],f[j] + 1);
    }
    for(int i = n;i >= 1;i--)
    {
        g[i] = 1;
        for(int j = n;j > i;j--)
            if(nums[i] > nums[j]) g[i] = max(g[i],g[j] + 1);
        res = max(res,f[i] + g[i] - 1);
    }
    cout << res;
    return 0;
}

  • 合唱队形

题目分析

求最少出列同学的个数，我们可以将其转换为合唱队形中同学数目的最大值，而合唱队形的要求则与上题一致，因而得解。

示例代码

#include<iostream>
#include<cstring>
#include<algorithm>

using namespace std;

const int N = 1010;
int f[N],g[N],nums[N];
int n,res;

int main()
{
    cin >> n;
    for(int i = 1;i <= n;i++) cin >> nums[i];
    for(int i = 1;i <= n;i++)
    {
        f[i] = 1;
        for(int j = 1;j < i;j++)
            if(nums[i] > nums[j]) f[i] = max(f[i],f[j] + 1);
    }
    for(int i = n;i >= 1;i--)
    {
        g[i] = 1;
        for(int j = n;j > i;j--)
            if(nums[i] > nums[j]) g[i] = max(g[i],g[j] + 1);
        res = max(res,f[i] + g[i] - 1);
    }
    cout << n - res;
    return 0;
}

2.先处理问题，转化为LIS

友好城市

题目分析

两条航线 (xi​,yi​),(xj​,yj​) 不相交 ⇔ (xi​−xj​)⋅(yi​−yj​)≥0

那么当我们将航线按某一坐标排序后（以 x 从小到大排序），两条航线 (xi​,yi​) , (xj​,yj​) 不相交 ⇔ (yi​−yj​)⋅(i−j)≥0 ，即非递减子序列。

因此我们只需要求排序过后的最长非递减子序列长度即可。

示例代码

#include<iostream>
#include<cstring>
#include<algorithm>

using namespace std;

typedef pair<int,int> PII;
const int N = 5010;

int n;
PII p[N];
int f[N];

int main()
{
    cin >> n;
    for(int i = 1;i <= n;i++)
    {
        scanf("%d%d",&p[i].first,&p[i].second);
    }
    sort(p + 1,p + n + 1);
    int res = 0;
    for(int i = 1;i <= n;i++)
    {
        f[i] = 1;
        for(int j = 1;j < i;j++)
        {
            if(p[i].second >= p[j].second)
            {
                f[i] = max(f[i],f[j] + 1);
            }
            res = max(res,f[i]);
        }
    }
    cout << res;
    return 0;
}

3.思维题（转化为LIS）

不同于上一类型，本类型主要侧重于通过数学证明等思维方法进行转化，难度更大。

拦截导弹

题目分析

第一问求最长非上升子序列长度，很简单。

第二问求该序列最少要被多少非上升子序列覆盖，答案即为该序列的最长上升子序列长度，这实际上是Dilworth 定理的应用：最少链划分等于最长反链长度

（但是我没看懂证明QAQ，暂且搁置争议）

（还有nlogn的做法，待我有空补上）

示例代码

#include<iostream>
#include<algorithm>

using namespace std;

const int N = 1010;

int n,f[N],g[N],s[N];

int main()
{
    n = 1;
    while(cin >> s[n]) n++;
    int res1 = 0, res2 = 0;
    for(int i = 1; i < n;i++)
    {
        f[i] = g[i] = 1;
        for(int j = 1;j < i;j++)
        {
            if(s[i] <= s[j]) f[i] = max(f[i],f[j] + 1);
            else g[i] = max(g[i],g[j] + 1);
        }
        res1 = max(res1,f[i]);
        res2 = max(res2,g[i]);
    }
    cout << res1 << '\n' << res2;
    return 0;
}
